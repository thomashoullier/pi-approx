# Pi approximation
`pi-approx` packages routines for computing arbitrary precision approximations
of pi.

## Usage
### Machin

## Dependencies
* `pi-approx`:
  * [`taylor-series`](https://github.com/thomashoullier/taylor-series)
* `pi-approx/test`:
  * [`rove`](https://github.com/fukamachi/rove)

## Test
Launch tests with:

```common-lisp
(asdf:test-system "pi-approx")
```
